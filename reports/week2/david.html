David's Week 2 Individual Report

What were your concrete goals for the week?

My concrete goals were to:
  • Set up graphics libraries (OpenGL, glm, glad, imgui)
  • Render basic wireframe cube on the client window
  • Add simple user interaction to move the wireframe cube around
  • Add camera interaction via mouse movement

What goals were you able to accomplish?

  • I worked with Anthony to get the graphics libraries set up on both our local devices and the lab computers. There were some issues with OpenGl versioning and differences between the operating systems we were using, but we were able to resolve them and have a client window show up and compile.
  • We added code for rendering a simple cube using OpenGl, but the cube is unable to render on the window.

If the week went differently than you had planned, what were the reasons?

  • The setup of the graphics libraries was more involved than originally thought, and there were some issues with getting all of the code to compile on the lab Windows machine. We had to import glew for the Windows machine instead of using glad, which ended up allowing the code to compile.
  • Due to the issue of the cube not rendering on the window, we focused more on trying to resolve this issue. In particular, the user interaction and camera movement require that the cube be visibly rendered on the window for us to test their functionality.

What are your specific goals for the next week?

  • For next week, the main goal is to get rendering working and resolve issues related to the rendering. This is the primary issue that blocked progress this previous week, and is critical to the game as a whole that rendering is able to run properly.
  • Afterwards, I aim to implement the user interaction, using both the cube and camera movement. While the cube may not be reused in the future, the implementation of movement, both of the camera and the object, can be adapted for player interactivity in later parts of the game.
  • The last goal would be to start tying the client with the server and adapting the client-side user interaction into events and communicate with the server. While this is dependent on the previous goal being completed, I hope to get the echo server functional so that we have a basic structure for communication between the client and server.

What did you learn this week, if anything (and did you expect to learn it?)

This week, I learned some aspects of using CMake for project compilation.

What is your individual morale (which might be different from the overall group morale)?
