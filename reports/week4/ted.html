Ted's Week 3 Individual Report

What were your concrete goals for the week?

  • Further implement different events (complex movement, collision) that may happen between the server and the clients
  • Finish refactoring the gamestate so that it can be used in all other parts of the project

What goals were you able to accomplish?

  • Gil, Tyler, Edward, and I were able to sit down together and finish the crux of our new gamestate, separating both the server and the client side. This allowed us to keep the logic and computation on the serverside, and only send down necessary data for the clients. While doing so, we also created Objects / SharedObjects to differentiate the data needed on the server and the client.
  • I also implemented the movement for the players along the 2D plane using keydown and keyup events, instead of constantly sending moveEvents. This allowed optimization for networking and built up the foundation for easier match with animations further down the line.

If the week went differently than you had planned, what were the reasons?

  • Since the graphics are not completely integrated with new objects and Gil is working on developing wall/maze, I am waiting on implementing and testing more collisions amongst the objects.
  • Jumping has been implemented but has not been generalized to platforming on different levels of y-axis. We would need further discussions about what types of movement the players are going to have.

What did you learn this week, if anything (and did you expect to learn it?)
