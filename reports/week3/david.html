David's Week 3 Individual Report

What were your concrete goals for the week?

My concrete goals were to:
  • Achieve milestone 1, which was to connect the client and server together for baseline interaction
  • Refactor client code & graphics rendering logic
  • Implement camera movement (WASD / mouse)

What goals were you able to accomplish?

  • As a group, we were able to achieve milestone 1 and connect the client and server with one another. When the graphics branch was merged, we had to go through and fix any merge conflicts, as well as a few issues that arose during the merge.
  • I was able to refactor some of the client code, but did the work on an older version of the development branch, so I needed to merge it into my feature branch to stay up-to-date. However, this requires some additional refactoring to account for all of the changes and additions to the development branch.

  • Because the refactoring is being delayed due to the merge, I was unable to implement camera movement, but did research on methods and potential organization.

What are your specific goals for the next week?

  • I aim to finish the camera interaction next week. This will be a critical part of the game to get first-person interaction working, and we will be able to build off of it for the final product.
  • I also aim to finish the refactoring. While the render loop works as currently implemented, I want to refactor and make the code easily extendible so that any future changes that we make will be easy to apply and will not have to completely break the existing code.
